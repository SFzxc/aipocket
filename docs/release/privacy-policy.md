# AIPocket Privacy Policy

Last updated: 2026-06-11

AIPocket is a free browser extension that lets users connect websites to approved AI providers without exposing real provider API keys to those websites.

## Data Stored By The Extension

AIPocket stores provider settings, provider API keys, approved website sessions, allowed models, session expiration times, and optional request limits in Chrome extension storage.

## Data Shared With Websites

Websites do not receive provider API keys. Approved websites receive session metadata, approved model names, and streamed model output for requests they initiate through AIPocket.

## Data Sent To AI Providers

When an approved website starts an AI request, AIPocket sends the request input, selected model, and required authentication headers to the selected AI provider endpoint. Provider responses are streamed back through the extension to the requesting website.

## Data Selling And Advertising

AIPocket does not sell user data. AIPocket does not use user data for advertising.

## User Control

Users can remove providers, revoke website sessions, clear extension storage through Chrome, or uninstall AIPocket to remove extension data.

## Payments

AIPocket is free. It does not process payments, provide billing, or sell AI API usage.
